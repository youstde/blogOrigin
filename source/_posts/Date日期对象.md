###	1.获取指定的日期

```javascript
// ==================可以获取指定的日期===================
  var ss = new Date( 2016,10,0 );
```

### 2.获取指定月份的天数

```javascript
// ==================可以获取具体月份的天数===================
  var ss = new Date( 2016,10,0 ).getDate();   //31
```

### 3.将时间戳变成日期格式

```javascript
// =================将时间戳变成日期格式=====================
  var tt = Date.now(),
      result = new Date( tt );
```
